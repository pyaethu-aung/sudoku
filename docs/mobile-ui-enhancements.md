# Mobile UI Enhancements

## 1. In-app number pad (highest impact)

Replace the OS keyboard with a fixed digit pad (1–9 + erase) rendered below the board. The OS keyboard slides up and covers the board; an in-app pad keeps the board fully visible and feels native to Sudoku.

## 2. Same-digit highlighting

When a cell is selected, highlight all other cells on the board that contain the same digit. Standard Sudoku UX — helps the user spot patterns quickly.

## 3. Erase button

With an in-app pad, erase becomes a dedicated, always-visible button rather than the ⌫ key buried in the OS number pad.

## 4. Keyboard / cell dismiss on outside tap

Tapping outside a cell (on the board background or scroll area) should deselect the active cell and dismiss the keyboard. Currently neither happens.
